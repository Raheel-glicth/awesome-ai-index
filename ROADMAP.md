# Roadmap

This document outlines the development roadmap for **awesome-ai-index**.

## Q3 2025 - Foundation

- [x] Initial dataset: 130+ models, 40+ vendors, 13 benchmarks, 7 frameworks
- [x] Automated validation via GitHub Actions
- [x] HuggingFace dataset mirror
- [x] Kaggle dataset mirror
- [x] v1.0.0 release
- [x] Community infrastructure (SECURITY.md, CODE_OF_CONDUCT.md, issue templates)

## Q4 2025 - Growth

- [ ] Expand to 200+ models
- [ ] Add pricing data per model (cost per 1M tokens)
- [ ] Add multimodal capability flags (text, image, audio, video)
- [ ] Weekly automated data scraping pipeline
- [ ] REST API endpoint for live queries
- [ ] Interactive web dashboard at alphaoneindex.com
- [ ] Submit to awesome-lists aggregators

## Q1 2026 - Scale

- [ ] Expand to 300+ models, 80+ vendors
- [ ] Add historical benchmark tracking (time series)
- [ ] GraphQL API
- [ ] Python SDK package (`pip install awesome-ai-index`)
- [ ] Integration with LangChain/LlamaIndex for model selection
- [ ] Automated weekly newsletter

## Q2 2026 - Enterprise

- [ ] Enterprise API with SLA
- [ ] Custom vendor risk reports
- [ ] Compliance audit automation
- [ ] AWS Data Exchange listing
- [ ] Snowflake marketplace listing

## How to Influence the Roadmap

- Open a [Feature Request](https://github.com/alpha-one-index/awesome-ai-index/issues/new?template=feature-request.yml)
- Vote on existing issues with thumbs-up reactions
- Contribute via [Pull Requests](CONTRIBUTING.md)
